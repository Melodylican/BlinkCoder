###BlinkCoder使用的技术主要有：


    1.     JFinal框架
    2.     Velocity模板引擎
    3.     EhCache缓存
    4.     Maven构建
    5.     前台采用Bootstrap3.1
    6.     使用Git作为版本控制
    7.     后台编辑器自己写的，基于markdown的Bootstrap风格的编辑器
    8.     Lucene构建搜索模块
    9.     使用Nginx做反向代理和处理部分后台静态文件
